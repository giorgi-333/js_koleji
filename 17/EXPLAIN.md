
- [დავალებებზე დაბრუნება](README.md)
```js

Math.random() // 0-დან 1-მდე

Math.floor(Math.random() * 10); // 0-დან 9-ის ჩათვლით

Math.floor(Math.random() * 11); // 0-დან 10-ის ჩათვლით

Math.floor(Math.random() * 10) + 1; // 1-დან 10-ის ჩათვლით

Math.floor(Math.random() * (max - min) ) + min; // min-დან max-მდე

Math.floor(Math.random() * (max - min + 1) ) + min; // min-დან max-ის ჩათვლით

```
